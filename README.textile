h2. How to install ?

h3. Git clone in Eclipse

*/!\ Clone ActionBarSherlock, ViewPagerIndicator and BeTrains in the same parent folder /!\*
For example:
* ~/documents/git/ActionBarSherlock
* ~/documents/git/Android-ViewPagerIndicator
* ~/documents/git/BeTrains-for-Android

h4. ActionBarSherlock

# Git clone from: "https://github.com/JakeWharton/ActionBarSherlock":https://github.com/JakeWharton/ActionBarSherlock
# Create a new project with the *library/* folder
# Use Android *3.2* API
# Include *android-support-v13.jar* from Android SDK

h4. ViewPagerIndicator

# Git clone from: "https://github.com/JakeWharton/":https://github.com/JakeWharton/Android-ViewPagerIndicator
# Create a new project with the *library/* folder
# Use Android *1.6* API
# Include *android-support-v4.jar* from Android SDK

h3. Clone BeTrains repository in Eclipse

* "https://github.com/iRail/BeTrains-for-Android":https://github.com/iRail/BeTrains-for-Android

h3. Add libraries

In the *libs/* folder, copy:
# gson-2.0.jar: "http://code.google.com/p/google-gson/":http://code.google.com/p/google-gson/
# actionbarsherlock-plugin-maps-3.4.1.jar: "https://github.com/JakeWharton/ActionBarSherlock/downloads":https://github.com/JakeWharton/ActionBarSherlock/downloads

h3. In the properties of your BeTrains project in Eclipse

# In "android" menu, select Google APIs *4.0*
